# Junie agent request
This project is to develop a Plotly/Dash app providing guidance to percussionists. 
Begin by creating an app that can record audio from the player's
microphone until they stop the recording. Recordings can be saved, and saved 
recordings loaded instead of a new recording. It then displays the waveform of the 
recording with zoom and pan controls. There is an option to play back the recording. 
The app also has a metronome feature that allows the user to set a tempo, with
a low tone that starts each measure and a high tone on other beets. 
There are controls for the metronome speed and number of beets per measure, as well 
as the volume of the metronome, recording and playback.
Metronome tone points are indicated on the waveform graph, with low and high tones 
distinguished. Pulse analysis identifies the most prominent moment 
in time for each beet played, with the maximum number of beets per measure 
limited by the number of measure divisions. These pulse times are indicated on the 
graph. The metronome and pulse points are saved and loaded along with the audio. 
Rhythm patterns and technique analysis will be added later.
